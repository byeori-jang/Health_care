# 🏥 Health_care Data Processing Practice

이 레포지토리는 **의료·헬스케어 데이터 분석을 위한 전처리 및 EDA 연습 공간**입니다.  
공공 데이터 및 공개 의료 데이터를 활용하여  
데이터 정제 → 가공 → 분석 흐름을 반복적으로 연습하는 것을 목표로 합니다.

---

## 🎯 목적
- 의료 데이터의 **구조와 특성 이해**
- 결측치, 이상치, 범주형 변수 등 **전처리 역량 강화**
- 추후 의료 AI / 헬스케어 분석 프로젝트를 위한 기초 구축

# 🏥 Health_care Dataset Practice Repository

이 레포지토리는 **헬스케어·의료 데이터셋별로 전처리 및 분석을 연습하기 위한 공간**입니다.  
각 데이터셋을 독립적인 폴더로 관리하여  
데이터 특성에 따른 전처리 방법과 분석 흐름을 기록합니다.

---

## 🎯 목적
- 의료 데이터셋별 구조 및 변수 특성 이해
- 데이터셋에 맞는 전처리 전략 비교
- 헬스케어 데이터 분석 역량 체계적 축적
